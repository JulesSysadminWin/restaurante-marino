\# Restaurante Marino



Página web de práctica desarrollada con HTML y CSS.



\## Objetivo



Proyecto de laboratorio para practicar:



\- HTML y CSS

\- Git y GitHub

\- Nginx en Ubuntu

\- SSH/SCP desde Windows

\- Despliegue local hacia una VM Linux



\## Estructura



restaurante-marino/

├── index.html

├── menu.html

├── styles.css

└── assets/

&#x20;   ├── logo.svg

&#x20;   └── hero-marino.svg



\## Entorno de práctica



La web fue desplegada en una VM Ubuntu con Nginx y acceso desde Windows mediante VirtualBox NAT.
